# Steganos
Steganos is a command-line based toolkit written in vanilla Python that support various basic steganographic and cryptographic data hiding techniques.

# Techniques to Support
* Text steganography
    - ~~Morse code~~
    - Caesar cipher (and ROT-13)
    - Null cipher
    - 1337 speak
    - Typo steganography

* Image steganography
    - BPCS steganography

* Audio steganography
